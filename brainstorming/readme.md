There are many publicly available datasets pre-processed for you to explore. Below are just a handful of websites. You can explore these as you're thinking about your projects, but you should formulate your questions before examining the data you're interested in.

Google Dataset Search: https://blog.google/products/search/discovering-millions-datasets-web/

Kaggle: https://www.kaggle.com/datasets

Agorize platform for challenges (some include data, see below for a few examples): https://www.agorize.com/en

Riipen platform (some include data): https://app.riipen.com/search/projects?selected=pORZeYOq

Canada's Open Government Initiative: http://open.canada.ca/en

Statistics Canada: https://www.statcan.gc.ca/en/start 

US Open Data: https://www.data.gov/

City of Calgary open data: https://data.calgary.ca/

World Bank Data Catalog: https://data.worldbank.org/data-catalog

Microsoft R Applications Network's List: https://mran.microsoft.com/documents/data/

Data Portals: http://dataportals.org/search

Open Science Data Cloud: https://www.opensciencedatacloud.org/publicdata/

DataQuest: https://www.dataquest.io/blog/free-datasets-for-projects/

DataWorld: https://data.world/

KDNuggets: http://www.kdnuggets.com/datasets/index.html

Datasets on GitHub: https://github.com/caesar0301/awesome-public-datasets

BuzzFeed Data: https://github.com/BuzzFeedNews

Reddit/Datasets: https://www.reddit.com/r/datasets/

UCI machine learning repository: https://archive.ics.uci.edu/ml/index.php

Lists of online data sources: http://www.opensources.co/

Article with 35 open source data websites: https://www.forbes.com/sites/bernardmarr/2016/02/12/big-data-35-brilliant-and-free-data-sources-for-2016/#6899afedb54d

IBM Watson Analytics sample datasets: https://www.ibm.com/communities/analytics/watson-analytics-blog/guide-to-sample-datasets/
